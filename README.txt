Trevizio Online v2

Essa é a versão online tipo site com:
- login e senha
- admin e colaborador
- bloqueio remoto real
- multi-loja Bling
- 3 botões: Salvar / Abrir nota / Imprimir

Login inicial:

Como rodar local:
1. instale Node.js 18+
2. abra terminal na pasta
3. rode: npm install
4. copie .env.example para .env
5. rode: npm start
6. abra http://localhost:3000

Como publicar no Render:
1. suba essa pasta para um GitHub seu
2. crie um novo Web Service no Render
3. conecte o repositório
4. build command: npm install
5. start command: npm start
6. defina BASE_URL com o link do seu site

Importante:
- para bloquear colaborador de longe de verdade, todos precisam usar o mesmo site publicado
